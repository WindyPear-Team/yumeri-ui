# yumeri-ui

A simple and modern CSS UI framework.

## Installation

```bash
npm install yumeri-uiw
```

## Usage

Import the CSS file in your project:

```html
<link rel="stylesheet" href="node_modules/yumeri-ui/src/index.css">
```

Or using a CSS import:

```css
@import "yumeri-ui/src/index.css";
```

## Theming

You can switch between light and dark themes by changing the import in `src/index.css`:

```css
/* From */
@import "./themes/light.css";

/* To */
@import "./themes/dark.css";
```
